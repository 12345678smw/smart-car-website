红外避障逻辑

多方位障碍物识别与智能绕行流程

避障算法流程

1

传感器数据采集

5路红外传感器实时采集障碍物距离数据，传输至主控芯片进行处理

2

障碍物判断

根据传感器阈值判断障碍物位置（左/中/右）及距离远近

3

路径规划

基于障碍物位置，智能选择最优绕行方向（左转/右转/后退）

4

执行转向

通过L298N驱动模块控制电机，实现差速转向绕行

5

恢复直行

绕过障碍物后，传感器检测无遮挡，自动恢复直行模式

代码预览

SmartCar.ino 
#include <MotorDriver.h>
#include <ObstacleSensor.h>
#include <OLEDDisplay.h>

MotorDriver motor;
ObstacleSensor sensor;
OLEDDisplay display;

void setup() {
  motor.init();
  sensor.init();
  display.init();
  display.showWelcome();
}

void loop() {
  ObstacleData data = sensor.read();
  display.update(data);
  
  if (data.front) {
    motor.backward(200);
    delay(500);
    if (data.left) {
      motor.turnRight();
    } else {
      motor.turnLeft();
    }
  } else {
    motor.forward(255);
  }
}
